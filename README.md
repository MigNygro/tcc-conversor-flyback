# tcc-conversor-flyback
"Projeto de TCC: Desenvolvimento de Conversor Flyback e Automação de Dados.


Conversor Flyback CC-CC: Plataforma de Ensino e Controle Digital<p align="center"><img src="https://img.shields.io/badge/Status-Finalizado-success?style=for-the-badge"><img src="https://img.shields.io/badge/UFMS-Engenharia_Elétrica-blue?style=for-the-badge"><img src="https://img.shields.io/badge/License-MIT-yellow?style=for-the-badge"></p>Este repositório contém o desenvolvimento completo de um conversor estático tipo Flyback, projetado como ferramenta didática para o estudo de Eletrônica de Potência e Sistemas de Controle. O projeto integra hardware, firmware em C++ e análise de estabilidade em Python.📂 Estrutura de Apêndices (Sequência do TCC)O projeto está organizado para guiar o usuário desde o código-fonte até a aplicação prática em bancada:📑 [A] Firmware do MicrocontroladorPasta: /01_FirmwareCódigo-fonte para Arduino (ATmega328P).Implementação de controle PID a 62,5 kHz.Funções de segurança: Anti-Windup e Soft-Start.📊 [B] Memória de CálculoPasta: /02_CalculosDimensionamento de componentes magnéticos e semicondutores.Análise do ponto de operação para carga de $100\ \Omega$.Cálculos de indutância e ripple de tensão.🐍 [C] Script de SimulaçãoPasta: /03_SimulacaoJupyter Notebook em Python.Validação da resposta ao degrau.Ajuste fino dos ganhos $K_p$, $K_i$ e $K_d$.🛠️ [D] Esquemas e LayoutPasta: /04_HardwareProjeto completo desenvolvido no EasyEDA.Diagrama esquemático e layout da PCB.Lista de Materiais (BOM).🎓 [E] Roteiro de LaboratórioPasta: /05_RoteiroGuia prático para aulas experimentais.Procedimentos de segurança e checklists.Roteiro de observação em Malha Aberta e Malha Fechada.🛠️ Tecnologias UtilizadasC++/Arduino: Lógica de controle em tempo real.Python (Pandas/Matplotlib): Simulação e análise de dados.EasyEDA: Design de hardware e PCB.Normas ABNT: Estruturação da documentação técnica.🚀 Como Utilizar este ProjetoSimulação: Execute o script em /03_Simulacao para entender a dinâmica do controle.Hardware: Utilize os arquivos de /04_Hardware para montagem da PCB.Firmware: Carregue o código de /01_Firmware via IDE do Arduino.Aula Prática: Siga o roteiro em /05_Roteiro para medições com osciloscópio.📄 LicençaEste projeto está sob a Licença MIT.
